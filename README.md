# Armina Agao — Creative Portfolio

A professional, elegant black & white portfolio website showcasing editing projects, achievements, and creative work from STI College Cotabato.

## 🌐 Live Demo
> Deploy to GitHub Pages and add your URL here

## ✨ Features

- **Elegant Black & White** aesthetic with high-contrast visuals
- **Fully Responsive** — works on mobile, tablet, and desktop
- **Custom Cursor** with smooth trail effect
- **Scroll Reveal Animations** for all sections
- **Lightbox Gallery** — click any image to view full-size
- **Animated Skill Bars** with intersection observer
- **Marquee Strip** for dynamic text flow
- **Mobile Navigation** with burger menu
- **Parallax Hero Image**
- **Zero dependencies** — pure HTML, CSS, and JavaScript

## 🚀 How to Deploy on GitHub Pages

1. **Fork or clone** this repository
2. Go to your repo → **Settings** → **Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live at `https://yourusername.github.io/repo-name`

## 🛠 How to Customize

### Change Name & Info
Open `index.html` and update:
- Your name in the hero section
- Tagline and introduction text
- Contact information in the closing section

### Change Images
Replace files in `assets/images/` with your own photos.  
Keep the same filenames (`img01.png`, etc.) or update the `src` attributes in `index.html`.

### Change Colors
Open `style.css` and modify the CSS variables at the top:
```css
:root {
  --black: #0a0a0a;
  --white: #ffffff;
  --gold: #c9a84c;  /* accent color */
  /* ... */
}
```

### Add/Remove Sections
Each section in `index.html` is clearly labeled with comments. You can duplicate, remove, or reorder sections freely.
